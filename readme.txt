== Description ==

3D Cover Carousel is a cool plugin & Html5 based carousel plugin that has the ability to continuously rotate/slide a set of images with the familiar 'cover flow' effect. You can easily add this plugin in and displayed at your front side with 3D animation slider images.

###Features###
<ul>
<li>Edit images and image name.</li>
<li>Image name is used as alt tag for seo.</li>
<li>Preview your slider before use it.</li>
<li>Slider installation into theme is simple just add shortcode to theme or pages/posts.</li>
<li>changes to images height,width</li>
<li>Changes to slider speed is easy.</li>
<li>Admin can set slider as slide with arrow left and right arrow.</li>
<li>Admin can set slider as circular slider.</li>
<li>Support responsive admin layout</li>
</ul>

### Usage ###
<ul>
<li>Use of 3D-Cover-Carousel is easy after activating plugin go to 3D-Cover-Carousel menu.</li>
<li>You can manage images by Manage images menu.</li>
<li>You can set settings for this plugin using slider settings menu.</li>
<li>You can add this slider to your wordpress page/post by adding this shortcode to [cover_carousel_slider] OR you can add this to your theme by adding this code echo do_shortcode('[cover_carousel_slider]'); to your theme</li>
</ul>

== Installation ==

This plugin is easy to install like other plug-ins of Wordpress as you need to just follow the below mentioned steps:

<ul>
<li>upload 3D-Cover-Carousel folder to wp-Content/plugins folder.</li>
<li>Activate the plugin from Dashboard / Plugins window.</li>
<li>Now Plugin is Activated, Go to the Usage section to see how to use wordpress 3D-Cover-Carousel.</li>
</ul>
